# 26QGSummerCamp

## 仓库说明

本仓库用于提交和管理夏令营期间的个人笔记。

每个人只维护自己的目录，并将每周笔记上传到对应周目录下。

## 目录规范

1. 每个人在仓库根目录下创建一个属于自己的一级目录。
2. 一级目录名称使用本人中文姓名。
3. 个人目录下按周次创建子目录，如 `第一周`、`第二周`。
4. 笔记必须为 `.md` 文档，并放在对应周目录下。

## 目录示例

```text
26QGSummerCamp/
├─ 张三/
│  ├─ 第一周/
│  │  └─ Java基础.md
│  └─ 第二周/
│     └─ Spring学习.md
└─ 李四/
   ├─ 第一周/
   │  └─ 算法笔记.md
   └─ 第二周/
      └─ 数据库总结.md
```

## 注意事项

1. 只在自己的目录下新增或修改内容，不要改动他人的目录和笔记。
2. 新成员请先按自己的中文姓名创建一级目录，再按周次上传笔记。
3. 上传的笔记文件请统一使用 `.md` 格式。
